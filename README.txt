Name: Binary Search Trees: Inorder and postorder traversing(Assignment2)

Description: This program creates two binary search trees from a list of names given(toSearchIn.txt). nameBST.jav is the binary for storing 		     names, their keys, and their frequency of appearance in the list. surnameBST.java does a similar thing for surnames. Driver.java 		     is used for printing out the traversal output. 


Instructions: To compile the program, use the make command on the terminal.
	      

List of files: Driver.java, Node.java, nameBST.java, surnameBST.java
